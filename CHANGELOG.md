# Changelog

Notable changes to the **[Spork Audio](https://sporkaudio.com)** macOS app, newest
first. Each version's `.dmg` is attached to the matching
[GitHub Release](../../releases). (Web-app changes are tracked separately at
[sporkaudio.com/changelog](https://sporkaudio.com/changelog).)

## 0.2.0 — 2026-07-05

- Large sample libraries (10k+) now load smoothly: folders index in the background with a live progress count, so the app never hangs on a big drop — and if you quit mid-scan, everything indexed so far is kept.
- Browse deep folder trees with a new drill-down sidebar, and scroll through thousands of files in a single folder without lag.
- Faster, more relevant search, powered by a full-text index.
- Spork now lets you know when a new version is available.

## 0.1.0 — 2026-06-25

- First public release for macOS — signed and notarized by Apple, so it opens with a normal double-click.
- Browse, audition, and tempo-fit your whole sample library: drum breaks, one-shots, and loops.
- Detect BPM and musical key, Reveal in Finder, drag-and-drop import, and playlist export & import.

<sub>0.1.0's installer was originally self-hosted; from 0.2.0 on, every release's `.dmg` lives here on GitHub Releases.</sub>
